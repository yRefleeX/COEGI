# COEGI - Portal para Cadastro e Busca de Motoristas

> Status: Em Desenvolvimento ⚠️ <br>
> Versão: 0.0.1-b

## Descrição
COEGI trata-se de um site em desenvolvimento pelos estudantes do Campus Campinas com o objetivo de diminuir a burocracia na qual motoristas privados precisam passar para que possam divulgar ou conseguir prestar seus serviços, facilitando tanto para estudantes uma forma de obter um meio de transporte sem precisar procurar exaustivamente por outros meios, quanto para os prestadores de serviço já que não terão que gastar tanto tempo e nem dinheiro para realizar tal divulgação. A pesquisa assume uma metodologia de coleta de dados qualitativa no decorrer do seu desenvolvimento. A expectativa do projeto consiste em sanar defasagens relacionadas à disponibilidade do transporte, a qualidade referente a esses serviços e também servir como uma ferramenta de apoio às diversidades geográficas, culturais, políticas e práticas socioeconômicas nesta instituição.
